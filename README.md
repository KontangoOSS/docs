# Kontango Documentation

This repository holds the public-facing documentation for Kontango — an open-source platform for running your own private cloud on hardware you own.

**Live site:** https://kontangooss.github.io/site/

**Source of truth:** This repo is mirrored from Forgejo. Changes can be made on either side; pushes to `main` flow into the live site automatically.

## What's here

- **`concepts/`** — Core concepts in plain language. Read these first.
- **`self-hosting-guide/`** — A 10-chapter guide covering the practical journey from cloud to self-hosted.
- **`offerings/`** — Tiered guidance for different audiences (basic, intermediate, enterprise).
