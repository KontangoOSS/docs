# Core Concepts

This section explains what Kontango actually is, in plain language. Each page is short and answers one question.

These pages are written for **non-technical readers** — the people using Kontango, not the people running it. If you operate the platform, you'll find this material useful too, but the technical depth lives elsewhere.

## The shape of it

Kontango is open-source software for running your own cloud on hardware you own. Use what you have, host where it makes sense, pay for help when you want it, do it yourself when you don't. The platform doesn't care which way you go.
